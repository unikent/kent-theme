Quotation

A quotation can help to entice readers into an article or highlight a key topic as a feature.

Use to provide credibility through a featured expert, this may be academic staff or a student.

Because of Kent, I have much greater academic and geographic possibilities. The experience has been priceless.

Dara Blumenthal, PhD Sociology View Dara's profile

Quotation person image

This is optional, but it is recommended as it adds visual interest. Use a square image at least 300x300 pixels. Never use pixellated images. The shot must be clear and face-on with natural lighting and surrounding as to not resemble a mug shot or passport photo.

Use a short quote with a focussed point. Text must not exceed more than 145 characters (about 4 lines on desktop).

The quote can also have an optional link leading to relevant content, such as a profile.

Icons

Icons are defined using a font file (Kent font). Only use icons defined in the Kent font.

See also Buttons, Social media

View Kent font icons

  • Kent logos - block, horizontal and Ke
  • menu/navigation icons - hamburger, drop down chevron, close X, tick, add, subtract, chevron links, next/previous button chevrons, search icon, home icon
  • audio/video icons
  • key system icons - Moodle, Email, KentVision, LibrarySearch, Timetable/calendar
  • social media icons
  • other - information, exclamation, loading/refresh icons, favourite, map marker, location marker, account avatar, wheelchair access, happy/sad/neutral face
