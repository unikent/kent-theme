Primary generic corporate

The default colour scheme is based on the primary generic corporate brand colours

Kent logo blue #003882
Dark blue #05345c
Gold #937227
Soft yellow #f7f2e1
Soft blue #e8ebf2
Soft grey #f5f5f5
Burgundy #671115
Charcoal #171717
  • Kent logo blue - used for the Kent logo
  • Dark blue - used on section headers, headings and as a spot colour on certain design
  • Gold - use on certain design elements and meta data
  • Burgundy - this replaces the Dark blue for the postgraduate section only
  • Charcoal - used for body text

Panel backgrounds colours

These are used to give a highlight to panel backgrounds. These are primarily soft background colours. Occasionally a darker primary colour is used to create contrast.

  • Soft yellow (panel tertiary),
  • Soft blue (panel primary),
  • Soft grey (panel secondary)
  • Dark blue or Burgundy for postgraduate as per above (panel primary).
