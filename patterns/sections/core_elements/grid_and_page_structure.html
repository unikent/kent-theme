Grid

The website uses a responsive grid based on Bootstrap. Typically content expands from mobile from one column to two columns to four columns.

Some featured content will span the full width of the page, whereas other elements, such as body text and feature layouts, are contained within a maximum width.

Page structure

Every page on the website uses these global elements to create its basic structure.

Kent bar (audience links)

The Kent bar is horizontal bar at the top of each page which provides links to priority audiences.

The Kent bar also acts as a light branding element on internal content. Once logged in, users will have personalised content accessible from the Kent bar.

Global navigation

This comprises of the three main links at the top of the page - Research, Study and Explore, which expose the key user journeys.

By simplifying options, it makes the site easier for prospects to get to the most important information that they need.

Global search

This is the gold search button to the right of the global navigation links. Global search provides categorised results which autocomplete upon typing.

It will be context aware, so if search in a school site, that schools results will be prioritised. It will always have course results integrated into the search.

Global footer

The footer contains additional links which are important, but shouldn't get in the way of the key user journeys in the global nav. Some of the links are duplicated in the footer.

It also includes Open Days, Social media, Giving to Kent sections and affiliate logos. At the very bottom of the page are subsidiary links.

Screen reader skip links

These are invisible links which assist screen reader users to navigate the content more easily without having to trawl through repetitive layers of navigation. They allow a user to:

  • skip to content,
  • skip to site navigation,
  • skip to section navigation,
  • skip to footer.
