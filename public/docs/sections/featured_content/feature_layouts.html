Featured content is displayed different layouts to create variation, interest, rhythm and hierarchy. They are usued in conjunction with feature panels. They help to provide navigation and marketing.

Each single feature comprises of a heading, image (where suitable), short introductory text and a link.

The layouts sits within a containing panel which can have a background colour.

In order to keep the layout balanced and visually aesthetic, feature content needs to be roughly of equal length.

Heading

These must be short and specific. Keep to 2 lines maximum (on desktop) - maxiumum 50 characters.

Introductory text

Try keep text to 3 lines or 4 lines maxiumum (on desktop) - maxiumum 110 characters.

Image

Images use a landscape ratio of 3:2 (minimum size of 1024x683px).

If you do not have a suitable image, rather consider not using an image by using a different feature layout.

View guidelines for images.
